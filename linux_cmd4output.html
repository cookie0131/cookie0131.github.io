echo

一般形式
echo string

  • \c 不换行
  • \f 近纸
  • \t 跳格
  • \n 换行

read

可以使用read语句从键盘或文件的某一行文本中读入信息,并将其赋给一个变量.一般形式
read varible1 varible2
例子

cat

可以用来显示文件内容，创建文件，以及显示控制字符。一般形式
cat [options] filename1 filename2
例子

管道

可以通过管道把一个命令的输出传递给另一个命令作为输入。管道用竖杠 |表示。它的一 般形式为:
命令1 |命令2
