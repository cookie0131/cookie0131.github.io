manager to do

发表于 2015-09-17   |   分类于 管理   |  

####QA manager有以下项目

  1. 人员的管理：包含指导，发展，生涯规划等
  2. 人员的录用与解雇
  3. 和公司目标对齐
  4. 建立要求质量的文化
  5. 在限期内交出高质量的产品

####QA manager本身需要哪些技能

  1. coach
  2. developing
  3. communication
  4. collaboration
  5. motivation

目前觉得挑战或困难

  1. 后继者的培养
  2. 人员的发展
  3. 如何雇佣到好QA
  4. 如何让好的人才愿意继续待在团队中
  5. 如何在高质量与实现之间取得平衡
