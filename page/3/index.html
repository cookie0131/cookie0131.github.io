玩转Jenkins - 使用Ldap

发表于 2016-09-28   |   分类于 持续集成   |  

公司内部的系统需要统一走Ldap,使用同一套账号和密码登录，所以搭建的Jenkins也不例外，需要接入，搞了好几天都没有配置好，后来发现原因是公司的统一认证使用的是 Windows ActiveDirectories。
需要安装Active Directory plugin插件后才能正常使用。

配置方式

位置：系统管理 -> configure global security
选中启用安全，安装好Active Directory plugin插件后，在安全域中会有一个 active directory选项

填写好以后可以用域账号登录看看是否成功

备注

设置中有一块 test domain name 和 test domain controllers,这个有时候会报错，提示 dadadc.cn doesn't look like a valid domain name,有可能是这个域名无法访问，也可能根本就没有问题，所以一切以实际用账号登录测试下来的结果为准
